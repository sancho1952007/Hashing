# Hashing

## What Is Hashing?
<pre>
Hashing Means, Securing Your Data In A One Way Irreversable One Way Method
A Hash Can Not Be Dcoded Easily.
You Can Only Encode Text To A Hash. But, You Can't Decode It Again...
</pre>
